# C++ Technical Mastery (GFG Aligned) 
 
This repository is a professional-grade technical lab for sharpening C++ expertise based on the GeeksforGeeks curriculum. 
 
## Curriculum Mapping 
- /01-Fundamentals-and-Control: Logic, Operators, and Control Flow. 
- /02-Functions-and-Compound-Types: Modular code, Arrays, Strings, and Structs. 
- /03-Memory-and-Pointers: DMA, Memory Safety, and Smart Pointers. 
- /04-OOP-and-File-Handling: Object-Oriented Design and Data Persistence. 
- /05-STL-and-Advanced-Concepts: High-performance Containers and System Programming. 
 
## Environment 
- IDE: Code::Blocks / VS Code 
- Compiler: GCC (MinGW)
